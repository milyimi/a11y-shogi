# アクセシビリティ将棋アプリ - プロジェクト概要

## プロジェクト名
**a11y-shogi** - アクセシビリティに配慮した将棋アプリケーション

## ビジョン
視覚障害者や高齢者を含む全ユーザーが快適にAI相手の将棋を楽しめるアプリケーション

## コア機能

### 1. ゲーム機能
- **AI対局**: ユーザー vs AI による将棋対局
- **難易度選択**: 3段階の強さ設定
  - **初級**: 高いランダム性（ミスを含む）
  - **中級**: 1手先読み + 簡易評価 + ミス率
  - **上級**: 4手先ミニマックス + アルファベータ枝刈り + 詰み優先 + 大駒連携評価
- **セッション管理**: ユーザー登録不要、セッション別に対局管理
- **外部AI対局テスト**: python-shogi / USIエンジンとの対局検証（開発者向け）

### 2. アクセシビリティ機能
- **スクリーンリーダー対応**: ARIA属性を完全実装
- **キーボード操作**: マウス不要で全操作可能
- **高コントラスト**: WCAG AAA基準を満たす配色
- **音声フィードバック**: 指し手などの重要アクション時の音声通知
- **テキストによる盤面表示**: ASCII形式での盤面確認

### 3. ユーザー体験
- 自動保存とリジューム機能
- ゲーム履歴の表示
- 局面の検討機能

### 4. ランキング機能
- **勝利時のニックネーム登録**: AI に勝利したユーザーはニックネームを登録可能
- **難易度別ランキング**: 初級・中級・上級それぞれのランキング
- **匿名性の維持**: ユーザー登録不要、ニックネームのみで参加
- **ランキング表示**: 勝利回数、手数、対局時間などで順位付け

## 技術スタック

- **バックエンド**: Laravel 12.48.1
- **フロントエンド**: Blade テンプレート + Tailwind CSS 4.1
- **データベース**: SQLite
- **AI エンジン**: 
  - PHP Minimax (depth=4, 500K transposition table)
  - 外部エンジン: python-shogi 1.1.1, Fairy-Stockfish 11.1 (USI)
- **テスト**: Pest 4.3.1 (127 tests), Puppeteer E2E (58 + 30 + 74 + 59 = 221 tests)
- **言語**: PHP 8.3.27, Python 3.11

## アクセシビリティ基準
- WCAG 2.1 AA準拠（できればAAA）
- WAI-ARIA ガイドライン準拠
- 日本の障害者差別解消法対応

## ユーザー層
- 視覚障害者（全盲・弱視）
- 高齢者
- キーボード操作を好むユーザー
- 将棋初心者から中級者

## 成功指標
- スクリーンリーダー利用者による100%操作可能性
- キーボードのみでの全機能操作可能性
- 初級AIとの対局完了率 > 90%
- Lighthouse Accessibility スコア 90+
